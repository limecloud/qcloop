# qcloop 项目完成总结

## 项目概述

qcloop 是一个**程序驱动的批量测试编排工具**，用于自动化执行 AI 驱动的测试任务，支持多轮质检和自动返修。

**GitHub 仓库**：https://github.com/limecloud/qcloop

## 核心价值

- **不漏项** - 数据库 claim 机制，确保每个测试项都被执行
- **可追踪** - 完整的执行历史、轮次记录
- **多轮质检** - worker -> verifier -> repair 自动闭环
- **可视化** - React 前端实时展示批次状态
- **易用性** - CLI + Web 双界面
